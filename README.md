Tata Supply Chain Co. Ltd. – Database Management System
Project Overview

This project focuses on designing a Database Management System (DBMS) for Tata Supply Chain Co. Ltd. to automate and streamline procurement, inventory management, material inspection, and material issuance processes.

The database has been designed based on the requirements provided by the company and includes normalization, entity relationship modeling, and database schema design.

3. Normalization

The database design has been normalized up to Third Normal Form (3NF) to:

Eliminate data redundancy
Improve data integrity
Ensure efficient data management
Objectives
Maintain inventory details of raw materials and finished parts.
Track vendor information and supplied items.
Record procurement activities through Challans and GRRs.
Manage transporter details.
Maintain quality inspection records.
Handle material issuance through MIRs.
Improve accuracy and efficiency of supply chain operations.
